        I, Table - bảng
            1, cú pháp - cấu trúc: table > thead - tbody - tfoot
                                            tr > th/td 
            2, Các thuộc tính thường dùng: rowspan, colspan,...

        II, Form - biểu mẫu:
            1, Cú pháp - cấu trúc:
                form: + method
                      + action
            2, Các thẻ nhập liệu:
                input: type(text, number, date, password,...)
                select: option 
                textarea
                    *) Lưu ý: + với input thì mặc định type là text 
                              + các thuộc tính trong input, select, textarea 
                                    - name: quy định vùng lựa chọn
                                    - id, class, attr-custom: xác định element
                                    - value != placeholder và value khác nội dung của option 
            3, submit, reset form 
                element thường dùng: input/button với type tương ứng => page sẽ bị load lại
                                     button với type="button"        => page sẽ ko bị load lại
